functions of n(instance size)

classify algorithms according to their growth rate

definitions

  • $g(n)$ functions $f(n)$ for which $g(n)$ is an upper bound
    $O(g(n)) = {f(n) | f(n) <= c*g(n), c > 0}$
($f(n)$ grows no faster than $g(n)$)
  • g(n) a lower bound
    $\Omega(g(n)) = {f(n) | f(n) >= c*g(n), c > 0}$
    $f: N->R+$
    ->f must be non decreasing
    $\theta(g(n)) = O(g(n)) \cap \Omega(g(n))
        = {f(n) | c1*g(n) <= f(n) <= c2*g(n)}$
theta classifies functions according to their growth rate

relation: 3 principles

show that theta is an equivalence
relation

- reflexible    $$f(n) \in \Theta(f(n))
                c1*f(n) <= f(n) <= c2*f(n)$$
- symmetric     $f(n) \in (g(n))
                -> c1*g(n) <= f(n) <= c2*g(n)
                -> 1/c2 * f(n) <= g(n) <= 1/c1 * f(n)
                -> d2 * f(n) <= g(n) <= d1 * f(n)
                -> g(n) \in \Theta(f(n))$
- transitive    $if f(n) \in \Theta(g(n)) and g(n) \in \Theta(h(n))
                -> f(n) \in \Theta(h(n))$
                Proof: assumption$
                -> c1*g(n) <= f(n) <= c2*g(n) and c3*h(n) <= g(n) <= c4*h(n)
                -> c3*h(n) <= g(n) <= 1/c1 * f(n)
                -> c1*c3*h(n) <= f(n)
                -> c1*c3*h(n) <= f(n) <= c2*c4*h(n), c1*c3 > 0, c2*c4 > 0 $

so theta is an equivalence relation

\infty & -> f(n) \in \omega(g(n))
\end{cases}$$

L’Hopital’s rule case:
$$\lim_{n \to \infty} \frac{lg^k n}{n} = \lim_{n \to \infty} \frac{k \ast (\lg n)^{(k-1)}) \ast \frac{1}{nln2}}{1} \\
= \lim_{n \to \infty} \frac{k}{\ln2} * \frac{(\lg n)^{(k-1)}}{n}\\
= \lim_{n \to \infty} \frac{(\lg n)^{(k-1)}}{n} \\
case 1: k is an integer = \lim_{n \to \infty} 1/n = 0 \\
case 2: not an integer = \lim_{n \to \infty} \frac{(\lg n)^{(k-[k])}}{n} = 0$$

Greatest common divsor

Euclid’s Algorithm
gcd(x, y) = gcd(y, x mod y)

  • proof
    gcd(x, y) = gcd(y, x - ny), n = \floor x/y
    1) d = gcd(x, y)
    -> x = m1d; y = m2d
    -> x - ny = m1d - nm2d = (m1 - nm2)d
    -> d divides y and x-ny
    -> d <= gcd(y, x - ny)
    2) let d = gcd(x, x - ny)
    -> y = m1d; x - ny = m2d
    -> x = m2d + ny = m2d + nm1d = (m2 + nm1)d
    -> d divides x and y
    -> d <= gcd(x, y)
    therefore, gcd(x, y) <= gcd(y, x - ny) <= gcd(x, y), gcd(x, y) = gcd(y, x - ny)
    Lemma: if d divides x and y 
    s.t. d = \alpha*x + \beta*y, (\alpha, \beta \in integer)
<-> d = gcd(x, y)
    (\alpha, \beta, d) = Euclid(x, y)
    base case: x = \alphax + \betay
  • Algorithm Euclid(x, y)
    if y = 0
    return(1, 0, x)
    (\alpha’, \beta’, d) = Euclid(y, x mod y)
    return (\beta', \alpha' - \beta'\floor x/y)
    [d = \alpha’ + \beta’x mod y = \alpha’y + \beta’x - \beta’y\floor x/y
    = \beta’x + (\alpha’ - \beta’\floor x/y)*y]
  • Analysis
    1) y <= x/2, x mod y < y <= x/2
    2) x/2 < y, x mod y = x - y < x/2
    so x mod y < x/2, 1 bit smaller
    with every two recursive calls, the input is reduced by 1 bit
    -> 2n iterations -> O(n)
  • Euclid’s time complexity: O(n^3)

这标题不止是一场收尾战，是我整个赛季的体验。它还是有点浮夸，令我像一位女子高中生社团经理，即将跟着花泽香菜se、no地演唱起来，谁能想到我的竞技体育观众生涯热血很少孤独很多，主旋律是那几十首高登舔狗歌曲粤语填词。18-19赛季我对竞技体育的歪曲理解又加深了，甚至试图写这种赛季总结小论文：它本来应该庞大地占据我一整年的社交媒体，因此这只能当冰山一角；我也酝酿了一个月，因此这一晚上最大的contribution也就是把语言风格搞得平稳一点，不能像杜烧那样一看就肠子痛。

埃一年，温格离开后的第一个赛季，阿森纳气质正在蛇蜕式剥离，与我的蛇蜕式长大异轨同调，终于拉姆塞，酋长球场最后一位王储走了，dna只剩边边角角，串不成一条中轴线。温二期出走时大家可以骂咒球员，现在则骂不出口，只有守护克伦克，展望贡多奇能被练成约基奇，算了不敢展望，何况练成亨利也无法学人家单核带队。对不起我怎么又开始怨妇，总而言之本赛季最后一块石头掉下来，我们幸运地目睹了大厦坍缩全过程。这也是世界杯年，加上温格离任，球迷大规模更新换代，阿森纳在段子界的地位甚至都有所下滑。其实本赛季我有认识一些新的朋友，发现一些新的资源，但散场感还是非常强烈。我总觉得我应该去要一些朋友的联系方式，虽然没去但我有给一位初中时代的游戏亲友留言试图友谊续费（也没回复），因为第一次认识到她当年的真诚非常宝贵，没有得到我足够的珍惜，仅仅因为我的爱好与她不够重合。当然，散场靠人为不可避，是我早就明白的道理，这两句话和全文并没什么关系，只是我想借机表白亲友们（……）。我知道许多朋友算是一代人的团粉，潮起潮落，一浪三五新英雄*，这句话可以送给喜欢的选手们，球队却是座巨塔，永远站在海水里，不被高层骚操作拍矮，也不因载体干涸而搁浅（可能还是会干涸，我只是对比电子竞技）。

喜爱这样冰冷的事物体验当然不会好，我又极其擅长给自己制造焦虑，在半决赛之前购票，买完一秒当即进入贤者时间。这件事，即使做完我也不确定是否是我想做，很难想象看一场主队球赛是需要勇气才能办到的事，但我真的可以算是孤勇了。决赛名单尘埃落地之前我一直在做去看切尔西或者瓦伦西亚的心理准备，其实没有成功，与之相比，做决赛失利的心理准备要容易得多，但一周前看到姆希塔良去不了的那个夜晚，我仿佛瞬间回到上赛季的半决赛，奥巴梅扬无法上场，导致面对马竞梦幻开局，最终却无能狂怒收场。在飞机上，一旦我闭上眼，四周的旅客全部化为恶魔，我甚至开始担忧一次出师未捷的坠毁，也出于内心的恐惧写过两段预支游记，然而事实上，除了学会看财报或者剪视频之类无卵用的技能，观众得到的参与感是很少的，我们仅能做做心理准备，对改写剧本无能为力。我看球不太设闹钟，次回合非常精准地在两点五十分醒来（巴萨利物浦首回合我怎么就非常精准地在五点醒来，生气），那场球势大力沉，每一脚传球都砰砰作响，瓦伦西亚再度梦幻开局时，我对阿森纳的要求只剩晋级决赛，满足我的个人体验，让我明白我爱球队还是没有胜过自己，感到松了一口气。

阿森纳也确实绝不带来超额惊喜，复刻17年中国行——讲到中国行，我还为一场输球错过了今日的X战警宣传中国行。我没有像今天这样喜爱过球迷对立文化，也没有像同样今天，短短几小时后这样厌恶球迷文化，甚至发表了想对切尔西球迷进行杀戮的暴言，竞技体育真的激发人类兽性，划成扇形的观众席分区就是一个个笼子。本场比赛我印象最深的其实是下半场某次，阿森纳一个谁在己方禁区倒地，其他球员抱头下蹲，四周车蜜开始叫好。不同于以往的失利，这次真的不再有遗憾，也不再有空间无能狂怒，是非常正常的比赛，爱美丽不会搞些如梦似幻的美丽足球诱骗我们，一旦没有在开局看见优势，我们就知道胜利的概率去了七成，一旦在关键场次失一个球，我们就知道要崩，马上会有第二第三个。我也很崩，做不到继续应援——这时死忠区也已然开始图书馆，接下来的比赛除了写这个（……）最大的努力是表情管理，防止被导播选为幸运观众，惨案素材，我可是刚刚重温过巴西1：7时的球迷reaction集锦。

但要我说喜爱的源头是同样的易崩，或者“阿森纳就像你的人生”这类毒鸡汤，我又不很同意，要上纲上线一下：我这个人除去平等以外，对自由意志也有一点谜之偏好；遗憾的是，这二者都难以达成，我只有追求一些最为肤浅的自我选择，比如我无法把母语选为学术语言，也无法操控义务教育阶段分配老师同学，但是可以自己选一下实验室方向或者工作老板，至于选择再由什么驱动就难以讨论了。我一直将自己与阿森纳视为天作之合，直到2019年3月看了极度狂热那天我才意识到我对它的喜爱本身非常非常基于理性，它不是小时候被主场氛围就近染上的习惯，也不是某一场或一个人突发性闯进我的生活，它来自我漫长的选择*，热火朝天的西超欧冠我都体验过。之前有段时间暴言发表过多不小心吸到了一位支持本国国家队+本地俱乐部的朋友，他当时就略带嘲讽地提到，他这是最为正常的主队配置，尽管并不那么多见。确实我从小到大都很难对不是自己选择的东西有强烈归属感，我的球迷亲友多半像一位厂蜜朋友宁宁一样（对不起，大家都是厂蜜劳烦你实名出镜一下），对什么都有归属感，仙剑奇侠传蜀山琼华都要党争，不然就像赤木晴子一样相信球员。我只对阿森纳有，我是说，我也会希望加强我喜欢的游戏角色们或者division rap battle支持一下喜欢的组（横滨组！）之类，但阿森纳有点不一样，这个体验非常奇妙，我更容易把它和它的球员、团队、高层、球迷刻板印象分离开，而不容易分离遥远的我，仿佛一提到英超我就被降智。倒不是在骂别的亲友，是我真的不很习惯！（我要打个补丁，一想到这位本地球迷朋友都看到主队联赛冠了虽然是二级联赛，本豪门球迷还没有，就要感叹我迫真选股鬼才）

大部分时候，我试图使视角更全局一些，不擅长制造对立，就在刚才依然有闲心幻想自己是波切蒂诺，如何为对阵利物浦布局（但现实中还是不要胜利为妙）。如果将人生行进比作旅途，这样的代入感不在我计划内，更多地，是在我靠在巴士座椅上的后脑勺里忽然冒出一群人杂乱地拼抢，场面盛大。其他厂蜜朋友们代入时远比我坚定纯粹，善于自我认同，看极度狂热时认为像科林费尔斯一样比起自己把握住一次职业生涯的机会，更希望主队冠何等理所当然。这令我想起以前看过一个新媒体文风软文，讲头条老板在校时意志坚定，能执行自己围绕人生目标定下的行动，绝非需要头条这类玩物来沉迷才能勉强存活的大多数人。我没有到将主队夺冠看得高过自己的程度（也许吧，高过我也出于丢人不会承认），也不会将它考虑进我的学业职业生涯，但不得不承认我心志软弱，必须有亚文化爱好这样人世间的好东西来沉迷。直到十五岁我还在梦想未来逃到全息网游中存活，仅剩大脑最好，只是后来身处的环境让我越来越愿意参与世界、有点用处，也将过分高看主队夺冠或者想和纸片人老婆结婚视作丢人——我可非常尊重人类社会规律。但本质上，那不怎么舒适，我最大的舒适就是当一位全无输出的渺小观众，这个矛盾将贯穿我整个人生。我个人私密又羞耻，好东西们公开而美丽，竞技体育又如此生动，在众多好东西当中存活感非常强烈，不像别的，总教我害怕爱的本质是我的虚荣。我非常在意自己感情的本质。漫长的一年我一遍一遍质问自己，如今终于稍微愿意承认，我喜欢阿森纳比足球本身多一点（只多一点点！），已经开始阻碍我享受广阔的快乐。它是我逃避苦痛的港湾，也是我新的苦痛的根源，苦痛自有永恒的本质。在阅读各种别人的小论文时本自省爱好者一边歌颂他们口味各异的真情实感，一边感到忐忑，我自己的真情实感是否过溢了，这是否离事实有点远，我们是否并没有自认全宇宙最惨的权利，说工作学习被影响或者人生被改变是否在互相抬举。但有一点我是明白的，世界上最美好的生活，腰缠万贯，身心自由，智力进化，出任帝国科学技术总监，迎娶克林贡美女，大脑过筛多次后也只剩三顿饭一张床*，显得索然无味，而“十几年等待后主队夺冠”，以及与之同出一脉的心情，却是我的想象中根本搜索不到的，硬要打个比方，它们可能和极限运动心情也算同出一脉，但比那些长远又深刻得多。

我近期最喜欢的中文球迷小论文——这里也暨读后感一下——有两篇来自lofter云什么尾巴，我简称酪太太，一篇《予取予球》来自杜烧太太。还有一篇在她评论里看到的梁文道写的《失败是更多失败的母亲》我没有列为最喜欢，因为它非常文人酸腐，失恋轮回就语出此文引用的另一篇小论文。当然，我也属于那种“一群人中最酸腐一个”，因此我还是相当喜欢它：该绝望的地方我比他程度更甚，诸如发散到人生、或者想象自己死于一场赛后骚乱的事，尽管前面刚否认过，我现在就干着，毕竟我看阿森纳以来它从未稳固过，一直在烟消云散，哪日终于奖杯到手，分封诸侯，也总有不同的个体带着不同的遗憾（我因为这点从小就不喜欢jump漫画！），实难圆满。但我想我真的不是很害怕长达几十年、每年一次的杯赛出局、联赛泡汤，因为登高峰一秒就能让此前此后一切绝望与遗憾凝结成的什么巍峨东西决堤，即使肉眼里看不到了，我也有在等这一天哦。

酪太太写法布雷加斯的小论文则稳定赚取我的鳄鱼眼泪，真诚热烈，澎湃奔涌，还常常使用重复加深技巧，我像看到阿丽塔出演感情线时掏出心脏，屡次确认自己真的超爱少年少女真心。我的真心，实在抱歉，挣扎而乏味，顾虑重重，恐怕永远不会晶莹剔透。我很多次思考过公众人物和粉丝之间的关系，怀疑我作为观众是不是坐到了太前排：在社交网络发问，如何割舍为之懊悔的过往；写诗嘲讽人蜜同担；甚至在同人文中嘲讽自己，为假神祇充满市侩气息的戏剧冲突要死要活，举着打折的季票——我个人则只能举着腾讯体育会员——宣告入股球队。但杜烧太太讲竞技体育是当代最接近英雄传说的地方又非常得到我认同，我常常把我的狭隘归结于中文文化圈的局促，导致一些年轻的能量找不到出口，只好去械斗*——我已经是典型的被武侠养大的一代，虎扑直男日日文艺复兴，首页诸多体育博主勇士更是恨不能化作比特币投身中美贸易战，现在写这个我也想意念引用一句白马啸西风李文秀“都好偏不喜欢”.txt。如果要发掘一些不同之处，也非常轻易，杜烧这篇也引用了波波维奇“没有虽败犹荣”.txt，我觉得正统英雄故事是可以虽败犹荣的，毕竟人生那么宽广，而竞技体育那么狭窄，赛场与球员的个人生活可能比你我还要割裂。现在的我仍然年轻，急切，容易升腾而难以被平息，在漆黑里发热，在有三次元朋友的地方以头抢地，在更年轻朋友的面前展现滞涩的愤怒，但我知道激情减退的温和一天一定会到来，就像人一定会走过青春期的焦虑到达稳健的中年阶段（改而进行中年焦虑），无论是否过着想要的生活，无论主队什么景况。要讲得更过分点，我对人类感情并没太信任，哪天彻底不爱它了体验更佳。挺小的时候我就体感我对时间跨度大的剧情容易感动，在对探根求源的不倦热爱下，我猜这来自外力变化会导致人类变化，而当前人类对时间维度的掌控还是过于低下，我总要学会像对待一位幼驯染的恋人那样对待这些自己，享受它，抗拒它，并慢慢等它离开。

semantic-aware sky replacement

Posted on 2019-03-20 | In paper review

内容概括：

流程：sky segmentation - semantic search - sky replacement

FCN训练

输入：LMSun dataset: 15000 out-door images作为train samples，原本数据集中label合并为11个。1045 images作为testset。
评价指标：the pixel accuracy and intersection-over-union (IOU) ratio。
输出response map：{f_i^1, f_i^2, …, f_i^n}, n表示n个可能分类，i表示像素，主要是语义分析结果，较为粗略。后期用于构建semantic layout descriptor进行搜索。
细化方式：online refinement。

  1. Sky Segmantation
    FCN results -> online classifiers. 对相邻像素xi、xj，构建公式E(X)。以FCN结果作为先验来训练这个分类器，改变每个像素属于sky/non-sky的分类令熵最小。
    E(X)参数与项：λ1=λ2=λ3=1，λ4=100；Uc由该像素RGB+高斯混合模型得到，Ut由SVM得到，Uf由response map中sky那一项得到，V也来自response map。Uc、Ut只是令分割更精确，引入response map确保你分割的东西是天空。

  2. Sky Search
    标准：content & layout；论文中待search的图片约为450张。
    2.1 semantic layout descriptor
    过去使用global descriptors（GIST）。我们提出新的，优点是更具多样性：首先归一化response map得到一维向量H，然后进行类似spatial pyramid pooling method的操作，即把整个图片分为九份，对每一份的response map归一化得H，再加上前面对整个图片的H，最终得到descriptor为一个n*(s+1)的二维向量（s=9）。
    2.2 selection：选择合适的宽高分辨率。
    2.3 diversity：确保天空的多样性，based on the inner product of color histograms。

  3. Sky Replacement
    3.1 semantic-aware transfer: match foreground statistics
    提出新的：基于scene parsing，优点是smooth boundaries。提出语义转化公式，其中包含一个归一化response map的似然值，一个传递函数Tn。为消除噪声，只保留概率排前几名的interpolation（大概就是局部传递），对数值进行标准化，删除foreground object label（之前LMSun标签简化环节中把不要的标签合成这个label）。
    3.2 Tn的细节：基于是否在参考与输入图片中都存在regions with label n，分为两种情况如下：
    Matched region: luminance transfer（亮度转换），有公式；chrominance transfer（色度转换），有reference。
    Non-matched region: 计算从整个参考前景到原始前景的传递函数，依然是亮度有公式色度有参考。

实现思路（云）：

  1. 直接使用FCN训练图片训练集（手动修改label），得到response map。
  2. 基于这个response map训练二元分类器，用于图片的切割。
  3. 但查找图片时依然用response map，参考基于GIST descriptor的论文实现一个搜索算法，进行sky search，并对得到的结果进行过滤筛选。
  4. 图片替换：这部分参考一下已有的自动替换代码？输入是第二步的结果。
  5. 语义自动转换：亮度转换部分按公式实现数学代码，复杂度不高；色度转换部分参考reference（我还没看，但在GitHub上有一个matlab实现的color transfer可供参考。）
