        = {f(n) | c1*g(n) <= f(n) <= c2*g(n)}$
theta classifies functions according to their growth rate

relation: 3 principles

show that theta is an equivalence
relation

- reflexible    $$f(n) \in \Theta(f(n))
                c1*f(n) <= f(n) <= c2*f(n)$$
- symmetric     $f(n) \in (g(n))
                -> c1*g(n) <= f(n) <= c2*g(n)
                -> 1/c2 * f(n) <= g(n) <= 1/c1 * f(n)
                -> d2 * f(n) <= g(n) <= d1 * f(n)
                -> g(n) \in \Theta(f(n))$
- transitive    $if f(n) \in \Theta(g(n)) and g(n) \in \Theta(h(n))
                -> f(n) \in \Theta(h(n))$
                Proof: assumption$
                -> c1*g(n) <= f(n) <= c2*g(n) and c3*h(n) <= g(n) <= c4*h(n)
                -> c3*h(n) <= g(n) <= 1/c1 * f(n)
                -> c1*c3*h(n) <= f(n)
                -> c1*c3*h(n) <= f(n) <= c2*c4*h(n), c1*c3 > 0, c2*c4 > 0 $

so theta is an equivalence relation

L’Hopital’s rule case:
$$\lim_{n \to \infty} \frac{lg^k n}{n} = \lim_{n \to \infty} \frac{k \ast (\lg n)^{(k-1)}) \ast \frac{1}{nln2}}{1} \\
= \lim_{n \to \infty} \frac{k}{\ln2} * \frac{(\lg n)^{(k-1)}}{n}\\
= \lim_{n \to \infty} \frac{(\lg n)^{(k-1)}}{n} \\
case 1: k is an integer = \lim_{n \to \infty} 1/n = 0 \\
case 2: not an integer = \lim_{n \to \infty} \frac{(\lg n)^{(k-[k])}}{n} = 0$$

= -(x^n/e^x) |(0->\infty) + n \ast \int_{0}^{\infty} x^{n-1} \ast e^{-x} dx \\
= -(x^n/e^x) |(0->\infty) + n \ast fact(n-1)$$

Cryptography: Operations on bits(number operations)

numbers: 100’s of bits long

  • Addition adding 3 1-bit numbers gives at most a 2-bit result(1+1=11)
  • Multiplication
    multi(x, y)
    if y = 0, return 0;
    if y is even, t = 2*mult(x, \floor y/2), return t
    if y is odd, return t + x
    Every recursive call, size of y reduced by 1.
    when the size of y is n, execute recursion O(n) times, Total O(n^2)
  • Division
    div(x, y)
    even: x = 2 \floor x/2
    odd: 2     \floor x/2 + 1
    O(n^2)

    Modular Arithemtic

    r = x mod y
    x \equiv y (mod N) <-> N divides (x - y)
    x is equivalent to y mod N
    x is congruent to y mod N

    properties:

    x \equiv x’(mod N)
    y \equiv y’(mod N)
    x + y \equiv x’ + y’(mod N)
    xy \equiv x’y’(mod N)
    commutative
    associative
    distributive
    Example: 2^345 \equiv (2^5)^69 \equiv 31^69 \equiv 1

    operations (mod N)

    0 <= x < N, 0 <= y < N; result: 0, 1, …, N-1
  • add(x, y): O(n)) # n = logN, the size of N
    s = x + y
    if s >= N
    s = s - N
    return s
  • multiply(x, y): O(n^2)
    (q, r) = div(x*y, N)
    return r
    largest value: O(n^2)

Greatest common divsor

Euclid’s Algorithm
gcd(x, y) = gcd(y, x mod y)

  • proof
    gcd(x, y) = gcd(y, x - ny), n = \floor x/y
    1) d = gcd(x, y)
    -> x = m1d; y = m2d
    -> x - ny = m1d - nm2d = (m1 - nm2)d
    -> d divides y and x-ny
    -> d <= gcd(y, x - ny)
    2) let d = gcd(x, x - ny)
    -> y = m1d; x - ny = m2d
    -> x = m2d + ny = m2d + nm1d = (m2 + nm1)d
    -> d divides x and y
    -> d <= gcd(x, y)
    therefore, gcd(x, y) <= gcd(y, x - ny) <= gcd(x, y), gcd(x, y) = gcd(y, x - ny)
    Lemma: if d divides x and y 
    s.t. d = \alpha*x + \beta*y, (\alpha, \beta \in integer)
<-> d = gcd(x, y)
    (\alpha, \beta, d) = Euclid(x, y)
    base case: x = \alphax + \betay
  • Algorithm Euclid(x, y)
    if y = 0
    return(1, 0, x)
    (\alpha’, \beta’, d) = Euclid(y, x mod y)
    return (\beta', \alpha' - \beta'\floor x/y)
    [d = \alpha’ + \beta’x mod y = \alpha’y + \beta’x - \beta’y\floor x/y
    = \beta’x + (\alpha’ - \beta’\floor x/y)*y]
  • Analysis
    1) y <= x/2, x mod y < y <= x/2
    2) x/2 < y, x mod y = x - y < x/2
    so x mod y < x/2, 1 bit smaller
    with every two recursive calls, the input is reduced by 1 bit
    -> 2n iterations -> O(n)
  • Euclid’s time complexity: O(n^3)

Testing for primes

suppose p is prime
\prod_{i=1}^{p-1} i \equiv \prod_{i=1}^{p-1} ai (mod p)
If N is composite(not prime)[and not a carmichael number],
-> \exists a \in {1, …, N - 1} s.t. a^(N-1) \not equiv 1 (mod N)

a^(N-1) \equiv c(mod N), c \noe 1
For each b \in {1, …, N-1} s.t. b^(N-1) \equiv 1(mod N)
(ab)^(N-1) \equiv a^(N-1)b^(N-1) \equiv c1 \not equiv 1(mod N)
