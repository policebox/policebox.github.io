  2. Linear Classification
    $$ f(x_i;W,b) = Wx_i+b $$
    x_i是竖着的（长    宽）的向量，W是（长宽）（类别数）的矩阵，类似分别在三种类别下的卷积核，b是偏置，得到一个代表不同类别分数的向量。
    有个人问怎么处理图片大小不同的数据集，答案是处理不了，先缩放。
    W是训练出来的，将每一行还原成图片就能看到双头马之类的东西。
    又有个人问how to deal with color problems，答案是根本解决不了，但神经网络可以看作堆叠的分类器，那个可以解决。

    SVM：计算所有比正确类别分高的score与正确分类score的差，求和。bug：W not unique，如果所有类别都是对的，无论如何对W缩放，最后lost function都为0.
    Softmax：soft在对[10, -100, -100]也很好处理

    lecture 4: Backpropagation and Neural Networks

    对所有参数求偏微分，得到一个vector，相当于更高效的gradient descent。
    L(θ)=sum(n=1,N){C^n(θ)} 其中C^n是对单个输入x^n的lost function
    L(θ)对一个参数w作偏微分，相当于C^n(θ)对w偏微分的和
    建议直接看李宏毅那个ppt。对每个neuron的输出z，有：偏微分 =
    frac{partial C}{partial z} * frac{partial z}{partial w}

    backward pass                   forward pass

    反向传播：类似倒推法，知道frac{partial C}{partial y}，层层推到z。相当于建一个反向nn，activation function为原来的导数。实现：雅各比矩阵

    Neural Network

    linear：f = Wx
    2-layer：f = W2*max(0,W1x)，其中max是激活函数。隐含层的好处是可以给汽车分类，如，隐含层的每一类代表一个汽车朝向等等。

      计算机运算能力并非无穷，don’t be overly ambitious.

      relu：可能会有relu在data cloud之外，成为dead relu，永远不被更新。原因可能是learning rate过高，导致它震荡。解决方法：Leaky relu、Prelu，令斜率可被学习。
